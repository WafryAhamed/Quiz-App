# backend-java

Spring Boot backend for authentication, PDF ingestion, AI quiz orchestration, quiz submission, and live quiz sessions.

## Run

1. Update `.env`
2. Ensure MongoDB is running
3. Run tests: `mvn test`
4. Start backend: `mvn spring-boot:run`

## Tech

- Spring Web
- Spring Security + JWT
- Spring Data MongoDB
- Spring WebSocket (STOMP)
- Lombok
