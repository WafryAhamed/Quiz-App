# mobile-app

Flutter mobile client for the Quiz Learning Platform.

## Features

- Login/Register
- Dashboard
- Upload PDF
- Quiz attempt + score
- Flashcards
- GPA Calculator
- Live Quiz join/create with real-time updates

## Run

1. Update `.env`
2. Install deps: `flutter pub get`
3. Analyze: `flutter analyze`
4. Run: `flutter run`
